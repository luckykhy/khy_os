# Product Hunt 文案草稿

**名称：** khy

**标语（≤60 字符）——选一条：**
- `The AI-native OS: a kernel, an agent, and a gateway`
- `Claude Code, a 16-backend gateway, and a real kernel`
- `A one-person AI-native operating system`

**描述（第一行是人们首先看到的）：**

khy 是一个「一次 `pip install` 即得」的 AI 原生操作系统：一个手写的 C
内核、一个 Claude-Code 级的智能体终端，以及一个前置了 16 个 AI 后端的网关
（自带 key，不锁定供应商）。

**Maker 的首条评论：**

Hi PH 👋 我基本是独自做出 khy 的，为的是回答一个问题：一个人，借助 AI，
能否可信地贯穿整个技术栈——从一个能在 QEMU 下启动的内核，一直到一个
「智能体*就是*界面」的终端？

今天你就能做到：
- `pip install khy-os`（或 `npm i -g @khy-os/khy-os`），运行一个带工具调用
  循环、权限门控、子智能体与目标模式的智能体终端
- 在 16 个 AI 后端之间路由，带自动故障转移，包括本地 Ollama
- 构建内核的可启动 ISO，并在 QEMU/VMware 中运行它

它是「源码可得」的（可免费运行/研究/非商业使用）。对局限性诚实以告：
内核仍是实验性的。非常欢迎反馈。

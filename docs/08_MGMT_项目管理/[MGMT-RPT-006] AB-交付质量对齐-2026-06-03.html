8.3 CW 排除安全性分析#

_toolSchemaConverter.anthropicToCW() 默认排除集: new Set(['web_search', 'websearch'])

  • Set.has() 是 大小写敏感 的
  • KHY 的工具名是 WebSearch (Pascal case)
  • Set.has('WebSearch') → false → 不被排除
  • 因此 KHY 的 WebSearch 可以安全转换到 CW 格式 (Kiro adapter)
